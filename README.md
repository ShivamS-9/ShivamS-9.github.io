# Shivam Singh - Personal Website

This is the repository for my personal portfolio website, showcasing my skills, experiences, projects, and contact information.

## Sections

1. **Home**: Introduction and tagline.
2. **About**: Overview of my academic background, interests, and hobbies.
3. **Skill Set**: Highlights technical skills with proficiency levels.
4. **Experiences**: Key professional and leadership roles.
5. **Projects**: Recent projects with brief descriptions.
6. **Contact Me**: Form and details to connect with me.

## Features

- **Responsive Design**: Optimized for all devices.
- **Portfolio Showcase**: Detailed sections for skills, experiences, and projects.
- **Contact Form**: Easy way to reach out for collaborations or inquiries.
